## Getting Started

First, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Used Technologies

- Zero Sync [https://zero.rocicorp.dev/](https://zero.rocicorp.dev/)
- ShadCN, TailWindCSS
- Varlock secret management via 1Password
- Biome linter
